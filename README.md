# Recipe Rating Analysis

Analysis of recipe ratings to understand what characteristics lead to highly-rated recipes. Project for DSC 80 at UCSD.

## Website

Visit the live website: [https://rishrav.github.io/recipe-rating-analysis/](https://rishrav.github.io/recipe-rating-analysis/)

## Setup Instructions

1. **Export Plots**: See `EXPORT_PLOTS.md` for instructions on exporting plotly plots from your notebook
2. **Add Data Tables**: Use `.to_markdown()` method on DataFrames to generate markdown tables
3. **Update Results**: Fill in actual numerical results (p-values, RMSE scores, etc.) in `index.md`
4. **Push Changes**: Commit and push to update the website

## Local Development

To preview the site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`
